# Mapa de Servicios - Gas / Agua

Una aplicación web interactiva para visualizar la disponibilidad de servicios de gas y agua en Lima, Perú.

## 🗺️ Acceder a la Aplicación

**Link directo:** https://anthonyobrasperu-lab.github.io/mapa-servicioss/

## ✨ Características

- 📍 **Mapa interactivo** con Leaflet.js y OpenStreetMap
- 🔍 **Buscador de direcciones** (Geocoder)
- 📏 **Herramienta de medici��n** de distancias
- 🔘 **Capas toggleables:**
  - Red de Gas
  - Agua / Desagüe
- 📱 **Geolocalización** - Botón "Mi ubicación"

## 🛠️ Tecnologías

- HTML5
- Leaflet.js
- OpenStreetMap
- Leaflet Control Geocoder
- Leaflet Measure

## 📄 Archivos

- `app_mapa_servicios.html` - Aplicación principal del mapa
- `index.html` - Página de inicio (redirección)
- `README.md` - Este archivo

## 🚀 Instalación Local

1. Clona el repositorio:
```bash
git clone https://github.com/anthonyobrasperu-lab/mapa-servicioss.git
```

2. Abre `app_mapa_servicios.html` en tu navegador

## 📝 Notas

- Requiere conexión a Internet para cargar los tiles del mapa
- Usa ubicación del navegador (requiere permisos)
